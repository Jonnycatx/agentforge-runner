How to Do It (Best Tools in 2026)
Use one of these (from current tools/docs):

py2app (Simplest, macOS-only, classic choice)
Bundles your Python script + embedded Python + deps into a standalone .app.
Steps:
pip install py2app
Create setup.py:Pythonfrom setuptools import setup

setup(
    app=['agent.py'],  # your main script
    name='WebDesignAssistant',
    options={
        'py2app': {
            'argv_emulation': True,
            'packages': ['langchain', 'openai', 'httpx'],  # add your deps
            'includes': ['tkinter'],  # if using GUI
        }
    },
    setup_requires=['py2app'],
)
Run python setup.py py2app â†’ outputs dist/WebDesignAssistant.app
Embed the .agentforge JSON (copy into app bundle or load from file).
For GUI: Use Tkinter (simple) or PyQt6 for modern chat UI (avatar, bubbles, input).
Distribute: Zip the .app or create .dmg (drag-to-Applications style) with create-dmg tool.


Briefcase (BeeWare) (Modern, cross-platform potential, great for native look)
Creates .app with embedded Python + Toga (native Mac widgets) for GUI.
Install: pip install briefcase
Init project: briefcase new (or migrate your agent.py).
Build: briefcase create macOS then briefcase build macOS then briefcase run macOS.
Outputs standalone .app bundle.
GUI: Toga gives native Mac feel (windows, buttons, text fields) â€” implement chat with labels, text input, scroll view.
Bonus: Future-proof for Windows/iOS/Android.

PyInstaller (Cross-platform fallback)
pip install pyinstaller
pyinstaller --onefile --windowed --add-data "config.json;." agent.py
Outputs .app on Mac (with embedded Python).
Use --windowed for GUI mode (no console).
Add --icon=your_icon.icns for custom app icon.
